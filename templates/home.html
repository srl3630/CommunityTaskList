Welcome to Community Task Center

This app allows users to post tasks to be done.

  • Users can post and see tasks using the tasks tab
  • Can claim tasks using the tasks tab
  • Can view claimed tasks on your dashboard as well as tasks you've submitted
  • Once you have completed the task, mark it as done using the dashboard
  • Once marked as done, the task is moved to the approve tab
  • Anyone besides the person who completed the task can approve the task, indicating they can verify the task has been completed
  • After being approved, the task is archived and a point is awarded to the person who completed it
  • Your score can be viewed using the leaderboard tab
  • Rack up those points and hope you have a good time with the website :)
